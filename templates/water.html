We took random images of stop signs. The thinking is to isolate stop signs is by color (for now). Using the OpenCV library in python, we convert the original image into HSV (for hue, saturation, value) image. We then assign lower and upper pixel values we want to isolate. We create a mask of this image. We then use bitwise AND operation on the original image and the masked image. This isolates the red pixels in the image and sets the rest of the pixels to 0 (for black). We then convert the red image into grey scale and binary image.
