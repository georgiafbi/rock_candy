Data analysis was done using the Python programming language and its libraries: scikit_learn, pandas, keras, numpy, and matplotlib. From the Figure 1.1, we can see that the most important component contributing to the compressive strength of concrete is water, cement, and age. We were able to achieve an accuracy score of 89% with a mean squared error (MSE) of 27.9 and mean absolute error (MAE) of 3.44. These are the best results when predicting the compressive strength of concrete using a Random Forest Regression (RFR) Machine Learning Model compare to other models tested, such as: Artifical Neural Network (ANN), Linear Regression + Polynomial Features (LR+PF), Support Vector Regression (SVR). The SVR model yielded an accuracy of 56%, LR+PF with 84%, and ANN with a MSE of 39.3 and a MAE of 4.98. The best parameters for our RFR model were determined using GridSearchCV library function that is a member of sklearn's model_selection package.
